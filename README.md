# Hotel Booking Website

A modern and elegant hotel booking website inspired by luxury resort experiences. Built using **HTML, CSS, and JavaScript**, this responsive website provides visitors with an immersive experience through stunning visuals, smooth animations, and user-friendly navigation.

Luviana Hotel Booking Website is live at https://sau123nethmini.github.io/hotel_booking/

## Features

### Luxury Landing Page
A visually appealing hero section with elegant typography, high-quality imagery, and smooth scrolling effects.

### Responsive Design
Fully optimized for desktop, tablet, and mobile devices to ensure a seamless user experience.

### About Section
Showcases the hotel's story, amenities, and hospitality experience.

### Amenities Showcase
Highlights premium facilities such as:

- Swimming Pool
- Spa & Wellness
- Fine Dining
- Room Service
- Free Wi-Fi
- Fitness Center

### Rooms & Suites
Displays available rooms with:

- Room images
- Room categories
- Pricing information
- "Book Now" buttons

### Restaurant Section
Presents dining experiences and menu highlights offered by the hotel.

### Testimonials
Customer reviews and guest experiences to build trust and credibility.

### Gallery
Beautiful image gallery featuring hotel life, rooms, restaurants, and outdoor attractions.

### Contact & Footer Section
Includes:

- Hotel information
- Social media links
- Newsletter subscription form
- Navigation links

### Smooth Animations
Interactive UI effects and transitions implemented with JavaScript for a premium feel.

## Tech Stack

- **Frontend:** HTML5, CSS3
- **Interactivity:** JavaScript
- **Design:** Responsive UI/UX
